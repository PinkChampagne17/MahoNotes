# 真步笔记 MahoNotes
一个简单的公主连结Re:Dive的计轴器。

![GitHub](https://img.shields.io/github/license/PinkChampagne17/MahoNotes)

未来也许会添加台服，甚至日服已经实装的角色。或者是永远也不会添加。
